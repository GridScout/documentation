## Invitation Steps

1. **Click the Invite Link**: Visit [GridScout Invite Link](https://discord.com/oauth2/authorize?client_id=1239680594539974787&permissions=274878417984&integration_type=0&scope=bot+applications.commands) to open the invite menu.

2. **Select Your Server**: From the dropdown menu, select the server where you want to add GridScout.

3. **Authorize Permissions**: Review the permissions requested by GridScout. These permissions are necessary for the bot to function properly.

4. **Complete Authorization**: Click "Authorize" to add GridScout to your server.

5. **Verify the Bot**: The bot should now appear in your server's member list and be ready to use!

## Required Permissions

GridScout requires the following permissions to function properly:

- **Read Messages/View Channels**: To see messages in channels
- **Send Messages**: To respond to commands
- **Embed Links**: To send embeds with F1 statistics
- **Use External Emojis**: For number, nationality and team emojis

## Optional Permissions

- **Mention Everyone**: Allows the bot to notify your members when an event is starting

## Getting Started

Once GridScout is in your server, you can start using it immediately with slash commands, such as:

```bash
/driver driver:Verstappen
/constructor constructor:Ferrari
/next
/standings
```
